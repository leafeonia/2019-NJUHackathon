## 项目简介

NJU教务助手安卓app

实现了登录、爬虫，可以在登录后基于教务网的信息进行一系列便捷的操作。

### 智能闹钟

根据第一节早课的时间和时间设置，点击“一键设置时间”，自动设置每天的闹钟时间。（目前的测试版会每隔一段时间自动唤起闹钟界面，即使程序在后台运行）


![](demo/clock1.png)

![](demo/clock_setting.png)

![](demo/clock2.png)

### 选课查询

显示所有的通识课的状态，并模拟抢课。

![](demo/course1.png)

支持按任意内容搜索

![](demo/course2.png)

### 学业总览

自动显示学业相关一些的信息，包括总学分数，各类课程的学分数、GPA以及各个学期的学分数、GPA

![](demo/overall.png)
